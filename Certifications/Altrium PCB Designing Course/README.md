# Basic PCB Designing Course

**Issued by:** Altium Education

## Overview

Completed the Basic PCB Designing Course provided by Altium Education to learn the fundamentals of professional PCB design.

The course introduces the complete PCB design workflow, from creating schematics to preparing a board for manufacturing.

## Topics Covered

- PCB design fundamentals
- Schematic capture
- PCB layout
- PCB footprints
- Component libraries
- Bill of Materials (BOM)
- Manufacturing workflow
- Basic ECAD concepts

## Why I Took This Course

After building several Arduino projects on breadboards and in Tinkercad, I wanted to learn how electronic products are actually designed for manufacturing.

This course gave me the theoretical foundation needed to begin designing custom PCBs.

## Next Step

I will be applying these concepts by redesigning my **Smart Assistive Navigator (SAN)** project as a custom PCB using **KiCad**, with the goal of producing a manufacturing-ready design.
